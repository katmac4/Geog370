Homework 6 part 2: Census data choropleth with a ratio

Katie Nicholls

Using data from the 2000 US Census, I made a map that shows the ratio of unmarried same sex couple households to the total number of unmarried households in the state of Georgia by county.

Choropleth with Equal Interval Classes

Equal interval classes split the boundaries of each class up into equal parts (ex.1-5,5-10,10-15) This makes the classes easy to read and understand, but if data is concentrated in one class, the true variability isn't shown.

Choropleth with "Pretty Break" Classes

Pretty break classes find natural breaks in the data to split up the classes. This is good for grouping data where there are distinct shifts in data, but if there are large outliers or no natural breaks in the data, it can misrepresent polygons.

Choropleth with Equal Quantile Classes

Equal quantile classes split the data up so that there are equal numbers of each entry in each class. This works well when you have data that are really close together, but doesn't work well when the data have large differences between them.
